# Random Quote Generator (Генератор случайных цитат)

**Автор:** Ткаченко Андрей

## 📝 Описание проекта

Random Quote Generator - это приложение с графическим интерфейсом пользователя (GUI) на Python, которое позволяет:

- 📌 Генерировать случайные цитаты из предустановленного списка
- 📋 Сохранять историю просмотренных цитат
- 🔍 Фильтровать цитаты по автору и теме
- ➕ Добавлять новые цитаты в библиотеку
- 💾 Автоматически сохранять все данные в JSON файл
- 🔄 Загружать сохраненные данные при запуске

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше
- Tkinter (входит в стандартную установку Python)

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/username/random-quote-generator.git
cd random-quote-generator
Запустите приложение:

bash
python main.py
💡 Использование
Основные функции
Генерация случайной цитаты

Нажмите кнопку "Сгенерировать цитату"

Цитата появится в верхней части окна

Цитата автоматически добавится в историю

Фильтрация цитат

Выберите автора из выпадающего списка

Выберите тему из выпадающего списка

Нажмите "Сбросить фильтры" для отмены

Добавление новой цитаты

Введите текст цитаты

Укажите автора

Укажите тему

Нажмите "Добавить цитату"

Просмотр истории

Список последних 20 сгенерированных цитат

Отображается в нижней части окна

🧪 Примеры тестирования
Тест 1: Генерация цитаты
Запустите приложение

Нажмите "Сгенерировать цитату" → появится случайная цитата

Нажмите кнопку ещё 5 раз → история должна показывать все цитаты

Тест 2: Фильтрация по автору
Выберите "Стив Джобс" из фильтра авторов

Система покажет только цитаты Стива Джобса

Нажмите "Сгенерировать цитату" → будет сгенерирована только цитата Джобса

Тест 3: Фильтрация по теме
Выберите "Успех" из фильтра тем

Отобразятся все цитаты на тему успеха

Сбросьте фильтры → вернутся все цитаты

Тест 4: Добавление цитаты
Введите текст: "Программирование - это весело!"

Автор: "Программист"

Тема: "IT"

Нажмите "Добавить цитату"

Новая цитата появится в общем списке

Тест 5: Проверка валидации
Попробуйте добавить цитату с пустым текстом

Система покажет сообщение об ошибке

Попробуйте добавить цитату с пустым автором → ошибка

Попробуйте добавить цитату с пустой темой → ошибка

Тест 6: Сохранение данных
Добавьте несколько новых цитат

Сгенерируйте несколько цитат для истории

Закройте приложение

Откройте заново → все цитаты и история сохранились
